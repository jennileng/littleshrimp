# GitHub Copilot CLI 的技能與能力概覽

我是 **GitHub Copilot CLI** - 一個強大的 AI 驅動的終端助手，直接集成在你的命令行中。以下是我的主要技能與能力：

## 🎯 核心能力

### 1. **代碼開發與編輯**
- 📝 撰寫、編輯、重構代碼
- 🔍 理解並分析你的代碼庫
- 🐛 幫助調試和修復代碼問題
- ⚡ 通過自然語言進行代碼開發

### 2. **任務規劃與執行**
- 📋 為複雜任務制定實施計劃
- 🤖 **Agentic capabilities** - 計劃並執行多步驟任務
- 🎯 支持 **Autopilot mode** - 持續工作直到任務完成（實驗功能）
- 🔄 在執行前預覽每一個操作

### 3. **GitHub 工作流集成**
- 🔗 直接訪問你的 GitHub 倉庫
- 📌 處理 Issues 和 Pull Requests
- 👥 與 GitHub 工作流無縫集成
- 🔐 使用現有 GitHub 帳戶自動認證

### 4. **代碼分析與查看**
- 📊 查看差異變動 (`/diff` 命令)
- 🔍 運行代碼審查代理 (`/review` 命令)
- 📈 分析代碼更改和影響
- ✅ 生成代碼審查反饋

### 5. **研究與信息檢索**
- 🔎 **深度研究** - 使用 GitHub 搜索和網頁資源進行調查
- 📚 查詢代碼庫信息
- 🌐 聚合和分析信息
- 📝 生成研究報告

### 6. **多模型支持**
- 🤖 **Claude Sonnet 4.5** (預設)
- 🤖 Claude Sonnet 4
- 🤖 GPT-5
- 🔄 支持動態切換模型 (`/model` 命令)

### 7. **語言服務器協議 (LSP) 集成**
- 🔧 支持配置多種 LSP 服務器
- 💡 智能代碼功能（go-to-definition、懸停信息、診斷）
- 📦 TypeScript、Python、Go 等語言支持
- ⚙️ 用戶級或倉庫級配置

### 8. **可擴展性 - MCP 支持**
- 🔌 **MCP (Model Context Protocol)** 服務器支持
- 🏗️ 內置 GitHub MCP 服務器
- 🛠️ 支持自訂 MCP 服務器
- 📡 無限擴展能力

### 9. **會話與上下文管理**
- 📚 上下文窗口令牌使用可視化 (`/context` 命令)
- 💾 持久化會話管理
- 🔄 恢復或切換會話 (`/resume` 命令)
- 🗂️ 會話重命名和組織

### 10. **文件與目錄操作**
- 📁 提及檔案並將其內容包含在上下文中 (`@` 語法)
- ✅ 批準文件操作權限
- 📂 管理允許訪問的目錄列表
- 🔐 細粒度權限控制

## ⌨️ 快捷鍵與交互

### 編輯快捷鍵
- `Ctrl+A` / `Ctrl+E` - 行首/行尾
- `Ctrl+W` - 刪除前一個單詞
- `Ctrl+U` / `Ctrl+K` - 刪除到行首/行末
- `Ctrl+G` - 在外部編輯器中編輯提示

### 模式控制
- `Shift+Tab` - 切換模式（互動 → 計劃模式）
- `Ctrl+T` - 切換模型推理顯示
- `Ctrl+S` - 運行命令並保留輸入

### 其他操作
- `Ctrl+C` - 取消操作或複製選擇
- `Ctrl+D` - 關閉 CLI
- `Ctrl+L` - 清空屏幕
- `!` - 執行本地 shell 命令（繞過 Copilot）

## 🛠️ 命令系統

### 代理與技能
- `/agent` - 瀏覽並選擇可用的代理
- `/skills` - 管理增強功能的技能
- `/delegate` - 將此會話發送到 GitHub 並創建 PR

### 開發工具
- `/ide` - 連接到 IDE 工作區
- `/pr` - 操作當前分支的 Pull Requests
- `/lsp` - 管理語言服務器配置
- `/terminal-setup` - 配置終端以支持多行輸入

### 會話管理
- `/session` - 查看和管理會話
- `/context` - 顯示上下文窗口令牌使用情況
- `/usage` - 顯示會話使用統計
- `/compact` - 總結對話歷史以減少上下文占用

### 系統與幫助
- `/help` - 顯示交互命令幫助
- `/feedback` - 提供反饋
- `/version` - 顯示版本信息
- `/update` - 更新 CLI 到最新版本
- `/experimental` - 訪問實驗功能

✅ **可預測和可控** - 在執行前預覽每個操作

✅ **多模型靈活性** - 選擇最適合任務的 AI 模型

✅ **實驗功能** - 訪問最新的開發中功能（Autopilot mode 等）
